# NI Simple Read Write Example using Ivi.Visa/

This example demonstrates how to use the IVI Visa pacakge to
perform simple synchronous I/O operations.  You can use this example
to send commands, such as "*IDN?\n", to your instrument.

## Tested implementations:

* IVI.Visa package 8.0.2
* NI 26.2 package 25.5.0.13

## Required Hardware

Any message-based device
